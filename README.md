## Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Android Studio / VS Code
- Git

### Installation

1. Clone the repository
```bash
git clone https://github.com/siraajul/submission.git
```

2. Navigate to the project directory
```bash
cd panels
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
